Steelwool Africa - Icon Generator

Instructions:

This tool generates icons for the PWA. Click each button below to download icons in different sizes required for the manifest.json file.

Note: All icons will be generated with the "SA" logo in Steelwool Africa colors.

After Download:

  1. Create an icons folder in your project
  2. Save all downloaded icons to the icons folder
  3. Icons should be named: icon-72x72.png, icon-96x96.png, etc.
